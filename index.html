More word-clouds!

Here you find instructions on how to create wordclouds with my Python wordcloud project. With the new and revamped interface, this is easier than ever. Compared to other wordclouds, my algorithm has the advantage of

  • filling all available space.
  • being able to use arbitraty masks.
  • having a stupid simple algorithm (with an efficient implementation) that can be easily modified.
  • being in Python ;)
